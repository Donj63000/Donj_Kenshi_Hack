# DonJ Kenshi Hack - dossier plugin

Ce dossier contient le coeur du plugin natif C++ pour le mod **DonJ Kenshi Hack**.

On y trouve notamment :

- le projet Visual Studio `DonJ_Kenshi_Hack.vcxproj`
- le code du terminal in-game
- le `SpawnManager`
- le `ArmyRuntimeManager`
- la specification `/army`

Pour la vue d'ensemble du projet, l'installation du mod, la compilation et les tests, voir le `README.md` a la racine du depot.
